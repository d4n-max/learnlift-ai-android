## What Changed Since v2

- Added Premium Study Pack foundation and preview flow.
- Added local starter content for SQL Interview Prep, QA Advanced, and Automation Testing Basics.
- Added coming-soon Premium pack metadata for Python Basics, JavaScript Basics, Business English, and Technical Interview Prep.
- Added AI Quiz Review and AI 7-Day Study Plan readiness.
- Added local AI usage limits for Free and Premium users.
- Improved Premium screen, RevenueCat package matching, and unavailable-product fallback copy.
- Added Premium gating rules that keep core Free study tools usable.
- Expanded content validation and content QA.
- Improved Smart Review, Adaptive Quiz, Progress, Settings, and Premium pack UX copy.
- Fixed the Progress `Start Adaptive Quiz` CTA so it opens Adaptive Quiz.

## What Changed Since v3.0

- Premium packs moved from foundation/metadata toward store-ready active packs for SQL, QA Advanced, and Automation Testing Basics.
- Premium pack UX now includes richer cards, count summaries, Premium badges, preview dialogs, coming-soon dialogs, and preview-limit messaging.
- Content QA now checks duplicate IDs, duplicate question text, correct answer IDs, option counts, difficulty values, topics, and preview expectations.
- AI Study Review and 7-Day Study Plan behavior is documented with compact, user-initiated payloads.
- Store, Data Safety, screenshot, paywall, Supabase, and release checklists are prepared for v3.6.
