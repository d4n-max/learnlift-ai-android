# LearnLift AI MVP Scope

## MVP Objective

Create a simple Android MVP for a general skill-building and study coach app. The first version should support multiple study paths through local JSON content while keeping the user experience focused and easy to build.

## Included In MVP

- Home dashboard
- Study path selection
- Daily study session
- Flashcards
- Quiz mode
- Progress tracking
- Local JSON content organized by study path
- Basic streak system
- Clean Jetpack Compose UI using the LearnLift AI brand colors

## Excluded From MVP

- Firebase
- Authentication
- Payments
- Live AI chat
- Backend admin panel
- Multi-language support
- Social features

## Initial Study Paths

1. English Vocabulary & Speaking Prep
2. Job Interview Prep
3. IT / QA Interview Prep as an optional sample path

## MVP Content

Initial content should be stored locally as JSON and organized by study path. Each path should be able to include flashcards, quiz questions, and daily study session content.

Suggested content areas:

- English vocabulary terms and speaking prompts
- Common job interview questions and answer guidance
- IT / QA interview basics as an optional sample path

## MVP Screens

### Home Dashboard

Shows the user's current streak, progress summary, selected or recent study path, daily session entry point, and quick access to flashcards and quizzes.

### Study Path Selection

Allows users to choose from available study paths. The first version should keep this simple and rely on local JSON content.

### Daily Study Session

Presents a small set of cards, prompts, or questions for the selected study path. The session should feel short, achievable, and habit-forming.

### Flashcards

Allows users to review terms, definitions, prompts, and study concepts for the selected path.

### Quiz Mode

Allows users to answer multiple-choice questions and receive immediate feedback.

### Progress Tracking

Shows completed sessions, quiz performance, path-level progress, and streak status using local app storage.

## Success Criteria

- A user can study without signing in.
- A user can choose a study path.
- A user can complete a daily session for a selected path.
- A user can review flashcards for a selected path.
- A user can take a quiz and see results.
- A user can see basic progress and streak information.
- The app feels clean, professional, friendly, and aspirational.
