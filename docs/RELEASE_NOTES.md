# LearnLift AI Release Notes

## Version 0.1.0 MVP

- Release type: Internal testing
- Package name: `com.learnliftai.app`
- Version code: `1`

## Summary

First internal MVP build of LearnLift AI.

## Included

- Study path selection
- Flashcards
- Quiz Mode
- Daily Study Session
- Local progress tracking
- Progress dashboard
- Settings
- Local starter content

## Current Study Paths

- English Vocabulary & Speaking Prep
- Job Interview Prep
- IT / QA Interview Prep

## Known Limitations

- Local-only progress
- No account login
- No cloud sync
- No live AI Coach yet
- No payments
- No analytics dashboard
- Starter content only
- Progress is global, not per study path
