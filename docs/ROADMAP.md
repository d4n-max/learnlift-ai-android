# LearnLift AI Roadmap

## Phase 1: Project Foundation

- Create initial documentation.
- Create Android project structure.
- Configure Kotlin, Jetpack Compose, and Material 3.
- Add brand colors and base theme.
- Establish simple MVVM conventions.

## Phase 2: Local Content Foundation

- Define local JSON content format.
- Organize content by study path.
- Add starter content for English Vocabulary & Speaking Prep.
- Add starter content for Job Interview Prep.
- Add IT / QA Interview Prep as an optional sample path.
- Load content from local app assets.
- Create basic models for study paths, topics, flashcards, quiz questions, and study sessions.

## Phase 3: MVP Core Screens

- Home dashboard
- Study path selection
- Daily study session
- Flashcards
- Quiz mode
- Progress tracking

## Phase 4: Local Progress And Streaks

- Store completed sessions locally.
- Track quiz results locally.
- Track progress by study path.
- Add basic streak calculation.
- Surface progress clearly on the dashboard.

## Phase 5: MVP Polish

- Improve empty states.
- Improve quiz feedback.
- Add simple first-run guidance if needed.
- Check visual consistency against the brand guide.
- Prepare for internal testing.

## Later Phases

- AI-assisted explanations.
- Remote content delivery.
- Driving License Exam Prep.
- School Exam Prep.
- Certification Prep.
- Software Testing Prep.
- Language Learning.
- Career Skills.
- Authentication, if needed.
- Payments, if needed.
- Backend admin tools, if needed.
